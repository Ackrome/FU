
## Лист 1: Генерация случайных ошибок (u₁...u₁₀₀)

**Что делается:**
- Генерируются 100 случайных чисел uᵢ для 5 экспериментов
- Эти числа имитируют случайные возмущения ε в регрессионной модели

**Зачем:**
- Чтобы создать «идеальные» данные, где все предпосылки Гаусса-Маркова выполняются
- Это базовый сценарий для сравнения с нарушенными условиями

---

## Лист 2: Расчёт теоретических значений Y

**Что делается:**
- Для каждого Xᵢ вычисляется Yᵢ = 5 + 7×Xᵢ + Uᵢ
- Где Uᵢ = uᵢ × σ — масштабированная ошибка

**Зачем:**
- Создать наблюдаемые данные, которые затем будут использоваться для оценки параметров модели
- Проверить, насколько точно МНК восстанавливает истинные параметры (a₀=5, a₁=7)

---

## Лист 3: Сценарий (б) — Нарушение нулевого математического ожидания

**Что делается:**
- К ошибкам добавляется константа: Uᵢ = uᵢ×σ + mm, где mm = 100
- Это нарушает предпосылку E(ε) = 0

**Зачем:**
- Продемонстрировать, что оценка свободного члена a₀ становится **несостоятельной**
- Показать смещение оценок при нарушении этой предпосылки

---

## Лист 4: Сценарий (в) — Гетероскедастичность

**Что делается:**
- Ошибки масштабируются по номеру наблюдения: Uᵢ = uᵢ × σ × i
- Дисперсия ошибок растёт с ростом i

**Зачем:**
- Показать, что при гетероскедастичности:
  - Оценки МНК остаются несмещёнными, но
  - Становятся **неэффективными** (не минимальная дисперсия)
  - Стандартные ошибки оцениваются неверно

---

## Лист 5: Сценарий (г) — Автокорреляция ошибок

**Что делается:**
- Ошибки генерируются по схеме AR(1): Uᵢ = d×Uᵢ₋₁ + c×uᵢ, где d = 0.2
- Это создаёт корреляцию между соседними остатками

**Зачем:**
- Продемонстрировать последствия автокорреляции:
  - Неверные стандартные ошибки
  - Некорректные t-статистики и доверительные интервалы
  - Потеря эффективности оценок

---

## Лист 6: Сценарий (д) — Нарушение экзогенности

**Что делается:**
- Вводится корреляция между X и ошибкой: Xᵢⱼ = Xᵢ + uᵢⱼ
- Экзогенная переменная «загрязняется» случайной компонентой

**Зачем:**
- Показать самый серьёзный тип нарушения:
  - Оценки МНК становятся **смещёнными и несостоятельными**
  - Даже при большом объёме выборки оценка не сходится к истинному значению

---

## Листы 7–10: Результаты 10 экспериментов для каждого сценария

**Что делается:**
- Для каждого из 5 сценариев проводится 10 независимых симуляций
- Фиксируются: оценённые коэффициенты, R², стандартные ошибки, статистики

**Зачем:**
- Оценить устойчивость результатов
- Показать вариативность оценок при разных реализациях случайных ошибок
- Сравнить качество оценок между сценариями
